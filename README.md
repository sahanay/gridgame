GridGame
====



Why another GridGame?
---------------------
Grids are all over the place. Why only use grids for layout when we can
use them to play a nice little fibinacci game?

```html
<!-- Create an element -->
<div id="app"></app>
```

```js
// instantiate new GridGame
var game = new GridGame({
  rows: 50,
  columns: 50,
  el: 'app'
});

// initialize game
game.init();

// => Have fun

```
